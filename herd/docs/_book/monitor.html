集成监控

集成监控

  • herd v1.1.2-beta.35 +

  • 可以通过设置环境变量 TERMINUS_AGENT_ENABLE 为 true 来开启监控功能。

  • 接入监控后建议在项目中直接安装 herd (使用全局 herd 有些本地安装的包是监控不到的)。
